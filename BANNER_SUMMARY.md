## 🚀 How to Use the Banner

### Option 1: Use it in this Repository
The banner is already added to your README.md and will display when people visit the repository!

### Option 2: Use it on Your GitHub Profile
1. Create a special repository with your username (e.g., `helloshanteria/helloshanteria`)
2. Add a README.md to that repository
3. Copy the banner image to that repository
4. Add it to the README: `![Banner](github_banner.png)`

### Option 3: Customize It
1. Edit `create_banner.py` to change:
   - Colors (modify the gradient colors)
   - Text (update titles and subtitles)
   - Tech stack (change the badge labels)
   - Visual elements (add/remove decorative items)
2. Run `python3 create_banner.py` to regenerate
3. View the new banner in `images/github_banner.png`
